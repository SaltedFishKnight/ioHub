ioHub（服务器）
1、将ioHub文件夹放入mods文件夹下
2、严格来说，这不是游戏Mod，它是独立于游戏程序的轻量级游戏服务器
3、使输出文字显示彩色
（1）推荐安装 Microsoft Store 的 Windows Terminal，并将 Windows Terminal 设置为默认启动
（2）不推荐ANSICON，它可能会导致游戏无法加载
（3）如果你的控制台已经支持ANSI Escape Sequences，或者你想要其他ANSI Escape Sequences支持，则忽略此设置
4、服务器配置文件路径：ioHub\run\config.txt
5、以下是你可能想要修改的设置
（1）-Xms2048m：等价于-XX:InitialHeapSize
（2）-Xmx2048m：等价于-XX:MaxHeapSize
（3）-DexternalPort=50100：ioGate连接ioHub服务器的端口，由于ioGate连接的端口范围限制在[50100, 65535]，若要更改端口，请将端口设置在[50100, 65535]之内
（4）-DbrokerPort=50000：ioHub的内部网关端口，一般不需要修改，只要不与externalPort和其他服务端口冲突即可
（5）-Dfile.encoding=GBK：设置为系统所使用的字符集
6、使用ioHub\run\launch.bat来启动服务器

ioHub (server)
1. Put the ioHub folder into the mods folder.
2. Strictly speaking, this is not a game mod, it is a lightweight game server independent of the game program.
3. Make the output text display color.
(1) It is recommended to install Windows Terminal from Microsoft Store and set Windows Terminal as default startup
(2) ANSICON is not recommended, it may cause the game can not be loaded.
(3) If your console already supports ANSI Escape Sequences, or you want other ANSI Escape Sequences supports, then ignore this setting
4. Path to server config file: ioHub\run\config.txt
5. The following are the settings you may want to modify.
(1) -Xms2048m: equivalent to -XX:InitialHeapSize
(2) -Xmx2048m: equivalent to -XX:MaxHeapSize
(3) -DexternalPort=50100: the port where ioGate connects to the ioHub server, since the port range of ioGate connection is limited to [50100, 65535], if you want to change the port, please set the port within [50100, 65535].
(4) -DbrokerPort=50000: ioHub's internal gateway port, generally do not need to change, as long as it does not conflict with the externalPort and other service ports
(5) -Dfile.encoding=GBK: set to the character set used by the system.
6、Use ioHub\run\launch.bat to start the server.
